## 3. 本组动态相对同学旧版的主要增强

### 3.1 车辆 / 行人

- 切线关键帧朝向，弃用 Follow Path 侧滑
- CW/CCW 双车道靠右；手动画路径库优先
- 路径消毒与密集关键帧，避免插值抄近路进人行道/城心

### 3.2 船只

- 优先 `SCG_Manual_Boat`（场景或 manual_paths.blend）
- 贝塞尔直接求值 + 贴水面；密集关键帧防穿城
- `distributed_boat_path_offsets` + `_safe_boat_path_offset` 避城心、均匀分布
- 速度约为原实现一半（`_MANUAL_BOAT_LOOP_PROGRESS = 0.09`）

### 3.3 手动画路径库

- Operator：`scg.export_manual_paths`（merge 保存，不覆盖库中未出现在场景的曲线）
- 命名：`SCG_Manual_Car_CW/CCW`、`SCG_Manual_Ped_*`、`SCG_Manual_Boat`

## 4. 建议验证步骤

1. 在 Blender 中启用 **`icity`** 插件
2. **Reload Scripts** → Start → 生成基础城市
3. 区域 6 → 添加车辆与行人 → 播放动画（靠右、不进人行道）
4. 添加中央山水 → 绘制/加载 `SCG_Manual_Boat` → **保存曲线到路径库** → 添加 5 艘船
5. 区域 5 → 载入示例 → 应用布局 → **恢复基础城市**（确认布局未破坏）
6. 区域 7 → 离线/LLM 指令如「添加2辆车辆」（确认 NL 仍可用）
