## 模块职责

| 模块 | 职责 | 对应阶段 |
|------|------|----------|
| `scg_register.py` | 统一 register/unregister | 一 |
| `scg_properties.py` | Scene 六项参数 + 生成后进入编辑 | 一 |
| `ui/panel_smart_city.py` | Smart City Generator 面板 | 一 |
| `operators/icity_bridge.py` | 安全 Start / 仅进入 Edit | 一 |
| `config/templates.json` | 模板编号 → 树/路/椅/天气等 | 二 |
| `core/assets_manager.py` | 2D/3D 文件路径 | 二 |
| `core/command_executor.py` | NL/JSON → handler | 四 |
| `core/city_generator.py` | Attribute / 节点 / 视口刷新 | 一～五 |
| `operators/generate_base.py` | 一键生成基础城市 | 一（已实现） |
| `operators/*` 其余 | 各面板按钮对应 Operator | 二～五 |

## 与 __init__.py 的集成方式（已实现）

1. `register()` 末尾：`from . import scg_register` → `scg_register.register()`
2. `unregister()` 开头：`scg_register.unregister()`（顺序与 register 相反）
3. `scg_register` 内依次注册：`scg_properties` → `operators` → `ui`
4. **不删除** 原有 `SNA_*` 类；新 Operator 通过 `bpy.ops.sna.*` 或 `city_generator` 复用 ICity 逻辑。
