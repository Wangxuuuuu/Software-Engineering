# 插件安装与运行说明

> 阶段二 Day 9 整理；与根目录 `README.md` 互补。

---

## 1. 安装

1. 复制整个 `icity` 文件夹到：

   ```
   Blender\<版本号>\scripts\addons\icity\
   ```

2. Blender → **编辑** → **偏好设置** → **插件** → 搜索 **icity** → 勾选启用。

3. 3D 视图按 **N** → **ICity** 选项卡 → **Smart City Generator** 面板。

4. 首次使用：点击 **Start ICity**，再使用生成/模板等功能。

**测试环境**：Blender 4.1+（`bl_info` 声明 4.0.1+）。

## 5. 视口与常见提示

- 道路纹理、路灯材质：使用 **材质预览** 或 **渲染**。
- 未 Start 时 Smart City 功能不可用（poll 检查 `ICity Base`）。
- 模板应用会调用 ICity 内部 `sync_city` 等 Operator，Info 面板可见日志。

### 看不到 Smart City Generator 面板？

1. **确认插件路径唯一**  
   正确位置只能是：
   `Blender\<版本>\scripts\addons\icity\`  
   不要在 Blender 安装根目录再放一份 `icity\`（例如 `D:\blender-4.1.0-windows-x64\icity\`）。  
   根目录旧副本会抢先被 Python 加载，导致 `scg_register` 未执行、面板不出现。

2. **确认加载的是带 SCG 的版本**  
   偏好设置 → 插件 → 应显示 **「ICity + Smart City Generator」**。  
   若仅显示旧版「ICity」，说明加载的不是团队扩展目录。

3. **在侧边栏 ICity 选项卡内向下滚动**  
   面板名为 **Smart City Generator**，位于 ICity Editor 等原生面板下方（`bl_order=5`）。

4. **重新启用插件**  
   取消勾选 icity → 保存偏好 → 再勾选启用 → 重启 Blender。

5. **查看系统控制台**  
   窗口 → 切换系统控制台；若注册失败会打印 `[ICity+SCG] Smart City Generator 注册失败` 及堆栈。
