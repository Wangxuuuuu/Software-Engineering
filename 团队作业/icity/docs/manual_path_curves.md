# 手动绘制三层道路轨迹（贝塞尔曲线）

当自动路径与场景不匹配（车辆斜穿、行人上机动车道、动画倒着走）时，可在 Blender 中手动画 **4 条闭合曲线**，插件会**优先**使用它们。

## 道路横断面（由里到外）

```
里层人行道  |  顺时针车道  |  逆时针车道  |  外层人行道
            ←—— 双向机动车道 ——→
```

## 1. 准备工作

1. 已 **Start** 并 **生成基础城市**。
2. 小键盘 **7** 进入顶视图，看清道路走向。
3. 大纲视图新建集合 `SCG_ManualPaths`（便于隐藏/显示）。

## 2. 必须使用的对象名称（区分大小写）

在 **物体模式** 选中曲线后，按 **F2** 重命名：

| 对象名 | 画在哪里 | 行驶/行走方向 |
|--------|----------|----------------|
| `SCG_Manual_Car_CW` | **顺时针**机动车道中心 | 沿曲线点序顺时针 |
| `SCG_Manual_Car_CCW` | **逆时针**机动车道中心 | 沿曲线点序逆时针 |
| `SCG_Manual_Ped_Inner` | **里层**人行道中心 | 双向共用 |
| `SCG_Manual_Ped_Outer` | **外层**人行道中心 | 双向共用 |
| `SCG_Manual_Boat` | **湖面**船只航线（闭合） | 沿曲线点序前进 |

- 顺/逆行人共用里/外两条曲线，插件自动分配方向。
- **船只**：画好 `SCG_Manual_Boat` 后点击「添加船只」，插件会沿曲线**均匀分布**多艘船（保持间距、避开城心）；无此曲线时回退为自动湖面环线。曲线可经面板 **保存曲线到路径库** 写入 `assets/manual_paths/manual_paths.blend`。
- **成对使用**：两条车道曲线都画好才用手动画；只画一条则该层仍用自动生成。
- 缺少某条时，该层回退为自动路径。

## 3. 绘制步骤（每条曲线）

1. **添加 → 曲线 → 贝塞尔**。
2. 进入 **编辑模式**：
   - **E** 挤出控制点，沿对应车道/人行道中心线点选；
   - **G** 移动控制点，**Alt+S** 可调整手柄使弯道平滑。
3. **曲线数据属性**（右侧扳手图标）：
   - 勾选 **循环 U**（或编辑模式 **Alt+C** 闭合）；
   - **分辨率 U**：12～24；
   - 关闭 **2D**（保持 3D）。
4. **高度 Z**：顶视图下可先随意；点击「添加车辆与行人」后插件会把曲线 **Z 对齐到道路顶面**。
5. **点序很重要**：
   - 顺时针车道：控制点按 **顺时针** 排列；
   - 逆时针车道：点序与顺时针 **相反**（或编辑模式 **W → 反向**）。

## 4. 验证与调试

1. **脚本 → Reload Scripts**（或重启 Blender）。
2. 再次点击 **添加车辆与行人**。
3. 时间轴 **空格** 播放：
   - 若 **倒着走**：编辑模式选中该曲线 → **W → 反向** → 重新添加；
   - 若 **偏左/偏右**：整体平移曲线到正确车道中心；
   - 若 **仍在机动车道**：检查是否误命名或只画了部分曲线。

## 5. 协作

曲线保存在 `.blend` 场景内，无需额外文件；团队统一命名即可。
